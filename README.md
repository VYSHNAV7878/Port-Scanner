# 🔍 Port Scanner

A fast, lightweight, and efficient command-line port scanning tool designed to identify open ports and discover services running on target systems.

## 🚀 Features
* **Targeted Scanning:** Scan specific IP addresses or network targets.
* **Speed & Efficiency:** Optimized to quickly check ports and return active statuses.
* **Clear Output:** Easy-to-read terminal output for quick analysis.

## ⚙️ Installation

1. Clone the repository:
   git clone https://github.com/VYSHNAV7878/Port-Scanner.git

2. Navigate into the project directory:
   cd Port-Scanner

## 💻 Usage

To run the port scanner, use the following command structure in your terminal:

python scanner.py <target-IP>
